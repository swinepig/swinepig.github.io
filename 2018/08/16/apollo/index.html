1.git clone apollo

git clone https://github.com/ctripcorp/apollo.git

一堆manven项目

需要使用的

apollo-configservice(eureka注册中心)

apollo-adminservice(eureka服务)

apollo-portal(eureka服务)

2.修改默认配置

执行SQL脚本apollo\scripts\sql\apolloportaldb.sql

apollo\scripts\sql\apolloconfigdb.sql(多环境多个)

apollo\scripts\build.bat或者build.sh
修改数据库连接配置包含config和portal

apollo对eureke服务进行了包装使用了meta server所以要在build.sh中修改实际的meta server url(其实就是eureka注册中心地址)

在ApolloConfigDB数据库中,找到表ServerConfig中的eureka.service.url配置项：UPDATE apolloconfigdb.ServerConfig SET ServerConfig.Value='http://localhost:9100/eureka/' WHERE Key='eureka.service.url';

3.编译-执行

build.sh/build.bat

4.部署

部署apollo-configservice

将apollo-configservice/target/目录下的apollo-configservice-x.x.x-github.zip上传到服务器上

修改scripts/startup.sh(这里的SERVER_PORT要和meta server url一致)

部署apollo-adminservice

将apollo-adminservice/target/目录下的apollo-adminservice-x.x.x-github.zip上传到服务器上

修改scripts/startup.sh

部署apollo-portal

将apollo-portal/target/目录下的apollo-portal-x.x.x-github.zip上传到服务器上

修改scripts/startup.sh

startup.sh修改LOG_DIR和SERVER_PORT(服务端口)
例子
LOG_DIR=/opt/logs/100003173
SERVER_PORT=9102

执行以上scripts/startup.sh即可

SERVER_PORT–eureka服务注册端口

5.访问测试

查看注册中心地址(meta server url配置地址)

登录portal页面 默认用户名密码为：apollo/admin
